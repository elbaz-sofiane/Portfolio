Bot Discord — API TMDb

Un bot Discord développé en Python qui interroge l’API The Movie Database (TMDb) pour fournir en temps réel des informations sur les films et séries : synopsis, note, année de sortie, et affiches directement dans un salon Discord.

✨Fonctionnalités

    ✅ Recherche par nom de film
    🎞 Titre du film
    📅 Année de sortie
    🏷 Genre(s)
    ⭐ Note moyenne
    📝 Synopsis
    🖼 Affiche officielle du film

    🚀 Démonstration

    Commande simple (discord) !film [Nom du film]
    👉 Le bot répond instantanément avec toutes les informations.

    🛠️ Technologies utilisées

    Node.js – backend du bot
    Discord.js – interaction avec l’API Discord
    TMDb API – pour récupérer les informations sur les films
    dotenv – gestion des tokens et clés API
    Railway – hébergement en ligne pour le bot

    Technologies

    Python, discord.py, requests, python-dotenv. Développement et debug réalisés sous Visual Studio Code.
